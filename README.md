# Career Compass (Modern React)

A fresh React + Vite multipage-style SPA for Career Compass with:

- Apple-like minimal UI with blurred glass cards
- Homepage scroll reveal animations
- Analysis page with RIASEC + Skills + MBTI selectors
- Suitable job opportunities output with direct job-search links
- RIASEC and MBTI test links
- About page with essential contact information

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
